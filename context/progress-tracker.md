## Architecture Decisions

- Using shadcn/ui on top of Tailwind CSS v4 (CSS variable approach, no tailwind.config.js).
- Dark-only theme: CSS custom properties defined in globals.css, mapped via @theme inline to Tailwind tokens. No light mode variables are defined.
- components/ui/* files are generated by shadcn CLI and must not be modified.
- shadcn semantic tokens (bg-card, bg-primary, text-muted-foreground, etc.) are wired to the Ghost AI dark palette via @theme inline in globals.css.
- Prisma v7 multi-file schema: generator/datasource in `prisma/schema.prisma`, models in `prisma/models/*.prisma`. Config via `prisma.config.ts`. Generated client at `app/generated/prisma/` — import from `@/app/generated/prisma/client`.
- `lib/prisma.ts` branches on `DATABASE_URL`: `prisma+postgres://` → `{ accelerateUrl }` + `withAccelerate()`; direct postgres → `PrismaPg` driver adapter.
